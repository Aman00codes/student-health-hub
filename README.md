## 🚀 Quick Start

### Prerequisites
```bash
python 3.9+
node 16+
PostgreSQL 13+
Redis
```

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/StudentHealthHub.git

# Backend setup
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Frontend setup
cd frontend
npm install
npm run dev
```
